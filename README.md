## Rock Paper Scissors Game

A simple Rock–Paper–Scissors game built using JavaScript. The user plays against the computer in the console.log

## Technologies Used
- JavaScript
- HTML

## Features

- Accepts user input (rock, paper, or scissors)
- Case-insensitive input (e.g., "rock", "ROCK", "RoCk" all work)
- Random computer choice
- Displays winner for each round
- Handles invalid input gracefully
